Dokumentasi API
Rental Kendaraan

REST API untuk sistem manajemen rental kendaraan. Mendukung metode GET, POST, PUT, DELETE. Tidak memerlukan API Key.

⚡ Base URL

https://nama-project-anda.vercel.app

Semua response menggunakan format JSON dengan struktur:

"meta": { "status": 200, "message": "...", "timestamp": "2025-03-17T..." },
"data": { ... }

👥 Pelanggan

MethodEndpointKeterangan
GET/api/pelanggan.phpAmbil semua pelanggan
GET/api/pelanggan.php?id=1Ambil pelanggan by ID
POST/api/pelanggan.phpTambah pelanggan baru
PUT/api/pelanggan.php?id=1Update pelanggan by ID
DELETE/api/pelanggan.php?id=1Hapus pelanggan by ID

Request Body (JSON)

namastringwajib (POST)Nama lengkap pelanggan

alamatstringwajib (POST)Alamat lengkap pelanggan
emailstringopsionalAlamat email pelanggan

🚗 Kendaraan

Request Body (JSON)

nama_kendaraanstringwajib (POST)Nama / tipe kendaraan

harga_sewaintegerwajib (POST)Harga sewa per hari (rupiah)

📋 Transaksi

MethodEndpointKeterangan
GET/api/transaksi.phpAmbil semua transaksi (dengan data JOIN)
GET/api/transaksi.php?id=1Ambil transaksi by ID
GET/api/transaksi.php?id_pelanggan=2Transaksi by ID pelanggan
POST/api/transaksi.phpBuat transaksi baru (auto hitung total)
PUT/api/transaksi.php?id=1Update transaksi by ID
DELETE/api/transaksi.php?id=1Hapus transaksi by ID

POST — Request Body

Request Body (JSON)

id_pelangganintegerwajibID pelanggan yang menyewa
id_kendaraanintegerwajibID kendaraan yang disewa
tanggal_sewastringwajibFormat: YYYY-MM-DD
tanggal_kembalistringwajibFormat: YYYY-MM-DD
status_bayarstringopsionallunas / belum lunas (default: belum lunas)

✅ durasi_hari dan total_bayar dihitung otomatis oleh server.

⚡ Status kendaraan otomatis berubah menjadi "disewa" saat transaksi dibuat, dan kembali "tersedia" saat dihapus.
